# L'Anatomia de la Fama en Codi Obert

Visualització de dades interactiva que analitza 5.113 repositoris de codi obert de GitHub per explorar quins factors estructurals expliquen la seva popularitat. A través de quatre preguntes de recerca, s'examinen la relació entre complexitat i watchers, la diversitat tecnològica per ecosistema, el rànquing de llenguatges per popularitat i l'existència de perfils estructurals mitjançant clustering k-means. Les visualitzacions estan implementades en D3.js i publicades com a pàgina web estàtica a GitHub Pages. Les dades provenen de GitHub Repos i GitHub Archive via BigQuery, i el dataset preprocessat és disponible públicament a Zenodo.

Treball final de l'assignatura de Visualització de Dades del Màster Universitari en Ciència de Dades de la UOC (2025–2026).
